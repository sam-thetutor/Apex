# Apex - AI Token Assistant

## Overview
Apex is an AI-powered token management application built for the Base blockchain. It allows users to send, receive, and swap tokens through natural language commands using Farcaster integration.

## Features

### 1. AI-Powered Chat Interface
- Natural language commands for all token operations
- GPT-4 powered intent parsing
- Context-aware responses

### 2. Wallet Integration
- Native Farcaster wallet support
- MetaMask fallback for standalone mode
- Secure wallet connection management

### 3. Token Operations
- **Balance Checking**: Query ETH, USDC, and DAI balances
- **Portfolio View**: See all token holdings with USD values
- **Send Tokens**: Send tokens to Farcaster users or addresses
- **Swap Tokens**: Exchange between supported tokens

### 4. Supported Tokens
- ETH (Native Base ETH)
- USDC (USD Coin on Base)
- DAI (Dai Stablecoin on Base)

## Usage

### Natural Language Commands

#### Balance Queries
- "How much ETH do I have?"
- "Check my USDC balance"
- "What's my portfolio?"
- "Show me all my tokens"

#### Send Operations
- "Send 100 USDC to @alice"
- "Transfer 0.5 ETH to @bob"
- "Send 50 DAI to 0x1234..."

#### Swap Operations
- "Swap 0.5 ETH for USDC"
- "Exchange 100 USDC for DAI"
- "Convert 50 DAI to ETH"

#### Help
- "help"
- "What can you do?"
- "Show me examples"

## Architecture

### Pages
- `/` - Homepage with hero section
- `/chat` - AI chat interface
- `/portfolio` - Portfolio overview
- `/transactions` - Transaction history
- `/help` - Help and documentation

### API Routes
- `/api/ai/parse-intent` - Parse user commands
- `/api/ai/agent` - LangChain agent for balance queries

### Components
- `ChatInterface` - Main chat UI
- `Navbar` - Navigation with wallet connection
- `HeroSection` - Landing page hero
- `MessageBubble` - Chat message display
- `QuickActions` - Quick action buttons
- `PortfolioOverview` - Portfolio display
- `TransactionList` - Transaction history

### Context
- `WalletContext` - Global wallet state management
